##### 方法

1. 构建文件目录
   ```
   .
   ├── node_modules 第三方安装包
   ├── public 公共文件
   ├── src 
   │   ├── api 存放系统公共请求
   │   ├── assets 系统所用文件
   │   ├── components 项目自定义组件
   │   ├── config 项目配置属性
   │   ├── hooks 项目hook存放
   │   ├── layout 项目启示页面
   │   ├── router 项目路由管理
   │   ├── store 项目状态管理
   │   ├── utils 项目工具
   │   ├── views 项目视图
   │   ├── App.vue 项目启动文件
   │   ├── main.js 项目运行程序
   │   └── style.css
   ├── index.html
   ├── package.json
   ├── vite.config.js
   ```
